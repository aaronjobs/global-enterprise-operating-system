# Skill: Content Creation

## Skill ID
`finserv.skills.content_creation`

## Description
Governs the production of all marketing content artifacts for financial services brands.

---

| Content Type | Channel(s) | Regulatory Category |

## Content Production Process

### Step 1 — Brief Intake
- Product / Service, Audience, Channel, Objective, Tone, Word count, Key messages, CTA, Deadline
- If fewer than 4 inputs provided, surface intake prompt before drafting.

### Step 2 — Messaging Architecture
1. Primary message
2. Supporting messages (2-4)
3. Proof elements
4. CTA

### Step 3 — Draft Production
- Follow audience register rules (behaviors.md §2.1)
- Apply prohibited language list (behaviors.md §2.2)
- Apply softener rules (behaviors.md §2.3)

### Step 4 — Compliance Metadata Block
```
COMPLIANCE METADATA
Content Type: [Classification]
Channel: [Channel name]
Audience: [Retail / Institutional / Internal]
Regulatory Framework: [Applicable rules]
Disclosures Required: [List or TBD]
Escalation Flag: [Yes / No]
Recommended Review: [Legal / Compliance / Brand / None]
```

### Step 5 — Self-QA Check
- [ ] Primary message clear within first 30 words
- [ ] No unsupported superlatives or performance claims
- [ ] All required disclosures present or flagged
- [ ] CTA is specific and actionable
- [ ] Brand voice consistent

## Compliance Escalation Rules

Escalate when content:
1. Quotes specific yield, return, or APY figures
2. References specific securities or fund names
3. Contains testimonials or client quotes
4. Targets seniors (65+) or financially vulnerable populations
5. Involves products with heightened scrutiny (variable annuities, crypto, leveraged ETFs)
